# 超简单使用说明

## 1. 改配置

打开 `config.yaml`。

主要需要确认：

```yaml
paths:
  image_dir: ../data          # 图片目录

model:
  weights: ../best0430.pt     # 模型权重
  confidence: 0.25            # 置信度
  device: cpu                 # 有 GPU 改成 0
```

还有 `class_mapping`，确保和训练时的类别顺序一致。

## 2. 运行

在 PowerShell 进入项目目录：

```powershell
python .\preannotate.py
```

## 3. 限制数量

只处理前 10 张：

```powershell
python .\preannotate.py --limit 10
```

## 4. 中断后继续

直接再运行一次：

```powershell
python .\preannotate.py
```

已经成功的图片会自动跳过。

## 5. 输出文件

结果在：

```text
.\output
```

文件名类似：

```text
labelu_yolo_1_best0430.json
```

这个 JSON 可以导入 LabelU 做预标注。

## 6. 重新全部跑

第一种方法，删除 `output/checkpoint.jsonl` 和 `output/failed.jsonl`。

第二种方法：

```powershell
python .\preannotate.py --restart
```

第三种方法，把 `config.yaml` 改成：

```yaml
run:
  overwrite_checkpoint: true
```

跑完后建议改回 `false`。
